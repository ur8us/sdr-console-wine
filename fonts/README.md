# SDR Console UI Font

`SDRConsoleUI.ttf` is a modified composite font generated by
`../tools/build-sdr-console-ui-font.py` from DejaVu Sans and Noto Sans Symbols
2. It is bundled only as part of this software and is installed only inside the
project's isolated Wine prefix.

The font is distributed under the SIL Open Font License 1.1 in
[LICENSES/OFL-1.1.txt](LICENSES/OFL-1.1.txt). It also includes material from
DejaVu Sans, whose required copyright, trademark, and permission notice is in
[LICENSES/DEJAVU-LICENSE.txt](LICENSES/DEJAVU-LICENSE.txt). The primary font
name is `SDR Console UI`; it does not use the names `Bitstream`, `Vera`,
`Tavmjong Bah`, or `Arev`.

SDR Console vendor binaries or vendor fonts.
